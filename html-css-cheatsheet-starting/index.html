My CSS Selector Cheat Sheet

by VectorXz

Basic

Selector Description Selected element example
HTML tag select element by HTML tag (Ex. h1,p,div) <p>
.class-name select element by class name <element class="class-name">
#element-id select element by element id <element id="element-id">

Advanced

Selector Description Selected element example
html-tag.class-name select element by HTML tag (Ex. h1,p,div) with specific class name <p class="class-name">
.class-name html-tag select child html element under an element with a specific class name <element class="class-name"> <p>Here</p> </element>
